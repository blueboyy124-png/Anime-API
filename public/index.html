Features

Built for developers

Everything you need to build an anime streaming website, mobile app, or Discord bot. No API keys, no rate limits, no hassle.

Full-text Search
Search by title, synonyms, genres, tags. Get autocomplete suggestions and paginated results via AniList GraphQL.
M3U8 Streaming
Get direct M3U8 streaming URLs from 12 providers. Supports sub, dub, download, skip timestamps, and thumbnails.
Rich Metadata
Characters, staff, relations, recommendations, trailer, stats, format, status, season, episodes, and more from AniList.
Advanced Filters
Filter by genre, tag, year, season, format, status, sort order. Multi-select support with AniList genre IDs.
Schedule & Calendar
View upcoming episodes with UNIX timestamps. Perfect for building calendar views and notification systems.
Fast & Cached
In-memory cache with 5-minute TTL and max 100 entries. Snappy responses with reduced upstream requests.
